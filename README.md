# restaurant-tips-analysis
Beginner data analysis project using Google Sheets
Restaurant Tips Analysis

This project analyzes restaurant tipping behavior using a public dataset.

Key steps:

* Calculated tip percentage
* Built pivot tables for analysis
* Compared tipping behavior across days
* Compared smokers vs non-smokers
* Created charts for visualization

Tools:

* Google Sheets

This project is part of my Data Analyst learning journey.
